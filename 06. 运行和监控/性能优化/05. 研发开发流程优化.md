## 优化前端开发团队的研发流程

1. 技术选型（页面渲染技术和混合式开发技术）

2. 项目初始化（React、Vue、Angular）

3. 依赖模块引入（私有 NPM）

4. 本地开发（自研前端工具）

5. 项目联调

6. 效果确认（产品、设计师确认）

7. 项目上线（部署系统）

## 前后端高效协作模式

1. 前后端指定数据接口

2. 前端页面制作

3. 启动交互实现

4. 后端开发

5. 前后端联调

6. 前后端上线

## 自动化测试

- UI 自动化：上手简单，稳定性差

- 接口自动化：稳定，性价比非常高

- 单元测试：性价比极高，一般有开发完成，但也有一些单元测试框架

## 自动化上线

1. 拉取代码库代码

2. 自动化工具在线打包编译

3. 代码上线部署到灰度机器

4. 代码上线部署至线上全量机器

5. CDN 后台静态文件更新缓存

## 提升团队整体代码质量

1. 指定代码规范（小组内或者团队内）

2. 静态代码扫描（可以在上线系统增加流程）

3. CodeReview（高工或者组内）

4. 阅读框架核心源码（站在巨人的肩上）

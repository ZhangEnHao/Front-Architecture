## 数据结构

数据结构是在计算机中组织和存储数据的一种特殊方式，使得数据可以高效地被访问和修改。更确切地说，数据结构是数据值的集合，表示数据之间的关系，也包括了作用在数据上的函数或操作。

## 逻辑结构与代码表现

- 栈（逻辑结构）可用数组（代码）表示

- 栈（逻辑结构）可用链表（代码）表示

- 队列（逻辑结构）可用数组（代码）表示

- 队列（逻辑结构）可用链表（代码）表示

- 树（逻辑结构）可用对象（代码）表示

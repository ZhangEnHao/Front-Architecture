## 概念

- 不同策略分开处理

- 避免出现大量 ``` if...else... ``` 或者 ```switch...case...```

```JavaScript
class User {
  constructor(type) {
    this.type = type;
  }
  buy() {
    if(this.type === "ordinary") {
      console.log("ordinary buy");
    }else if(this.type === "member") {
      console.log("member buy");
    }else if(this.type === "vip") {
      console.log("vip buy");
    }
  }
}

// test

let u1 = new User("ordinary")
u1.buy();
// ...
/********* 改造 *********/ 

class Ordinary {
  buy() {
    console.log("ordinary buy");
  }
}
class Member {
  buy() {
    console.log("member buy");
  }
}
class Vip {
  buy() {
    console.log("vip buy");
  }
}

// test
let u1 = new  Ordinary();
u1.buy();
// ...
```

## 设计原则验证

- 不同策略，分开处理，而不是混合在一起

- 符合开放封闭原则
